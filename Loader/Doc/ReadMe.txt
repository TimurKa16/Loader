Загрузчик программы
Он создан для возможности перепрошивки МК по SCI

Есть 3 сектора во FLASH:
	0x80000 - прошивка для загрузчика
	0xA0000 - прошивка для принятой прошивки
	0xB0000 - прошивка для заводской прошивки

В Загрузчике проверяется наличие новой прошивки в отдельной области RAM
Если есть прошивка, то прошивает
Если нет прошивки, то прыгает в основную прошивку

Настройка проекта:
ПКМ по проекту - Properties - Debug - Flash Settings - Erase Settings - Necessary Sectors Only (for Program Load)
